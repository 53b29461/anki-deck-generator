## 🎯 プロジェクト概要

TOEFL 3800 Rank 3の1,021語について、以下の高品質コンテンツを生成：

- **正確な日本語訳**: 文脈に応じた適切な意味
- **実用的な英語例文**: 各単語につき3つの実践的な使用例
- **記憶術・語源説明**: 効果的な暗記Tips
- **HTMLフォーマット**: Ankiで美しく表示される構造
